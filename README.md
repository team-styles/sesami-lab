## はじめに
「おともだちちょう」作成用リポジトリです。
Next.jsを主に使っており、スタイルにはTailwindCssを使用しています。
データベースについては検討中。

## 開発にあたって
### 環境
- node v20.11.0
- yarn 1.22.19

### setup
```
yarn install
```

### develop
```
yarn dev
```
-> access: https://localhost:3000/
ローカルサーバーが立ち上がり、Webサイトにアクセス
